About

Daryl is an open source theme for Jekyll designed to be performant, simple, and readable.

Documentation on how to use this theme is in the repo’s README. Jekyll usage documentation can be found at jekyllrb.com.

If you would like to contribute to this theme, please see the Development guidelines.
