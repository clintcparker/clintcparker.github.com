Hello World

Hi there, thanks for checking out Daryl, a performant and simple theme for Jekyll. I designed this theme to be as readable as possible, to be simple to use, and to keep the code used as vanilla as possible.

It’s open source for a reason: I’ve learned a lot from open source software and I’m trying to give a little something back. Hope you enjoy.

Welcome to Jekyll!

October 06, 2015

You’ll find this post in your _posts directory. Go ahead and edit it and re-build the site to see your changes. You can rebuild the site in many different ways, but the most common way is to run jekyll serve, which launches a web server and auto-regenerates your site when a file is updated.

To add new posts, simply add a file in the _posts directory that follows the convention YYYY-MM-DD-name-of-post.ext and includes the necessary front matter. Take a look at the source for this post to get an idea about how it works.
